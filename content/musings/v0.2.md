# Version 0.2 - Localization

Localization is too important to leave as an afterthought.
In many systems, it is added late and becomes difficult to maintain.
Musings takes a different approach by building localization into the core from the beginning.
In version 0.2, full content and asset localization is supported and easy.

## New: Localized file system

I created a localizing file system driver and routed every file operation through it.
The driver handles all localization, so the calling code does not need to be aware of it.
Because of this, localization applies consistently across content, templates, and assets 
without additional complexity.

A file named "filename.ext" can be localized by creating "filename.locale.ext".
If the locale is enabled in Musings' config, 
then it is considered to be the localized version of the file.

Locales have to be opt-in to avoid misinterpreting file names 
for users who do not wish to use certain locales.
For example, "program.cs.md" could mean documentation about the C# entry point,
or it could mean the Czech localization for a program document.
This ambiguity would be confusing for users who are not expecting localization.

## New: Translated strings table

A small translation table for common layout text was added to Musings.
The translations for the current locale are populated into a structure 
that can be accessed by the gohtml templates.

I considered adding full string localization, but I chose not to.
Musings is designed to keep localization simple and focused on file-based content.
The preset translations do not add new capabilities,
but they make common layout cases cleaner.

That list has not changed.

 * Version 0.3 will focus on indexing and searching.
 * Version 0.4 will add support for plugins.
 * Version 0.5 will focus on polishing and testing.

 * Version 0.7 will be a minor update with changes requested by a derivative project.
 * Version 1.0 will be the first stable release.

I'm excited and nervous about version 0.3.
It may be the most challenging version after this one,
but it will add so much cool factor to Musings.
